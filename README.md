# CEDPA Smart Supply Chain Analytics Platform

## Overview
CEDPA is an AI-powered supply chain analytics platform that provides:
- Demand forecasting
- Inventory analysis
- Risk prediction
- Business intelligence dashboards

## Features
- AI-based supply chain risk prediction
- Interactive analytics dashboard
- Demand forecasting
- Dynamic CSV upload
- Feature importance analysis
- Inventory insights
- Forecast report download

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
- NumPy

## Machine Learning
- Random Forest Classification
- Predictive Analytics
- Feature Importance Analysis
